Documentation for this section has not yet been entered.

Type Description
Affine Represents an affine transformation as implemented in GD. (I.e. an array of 6 doubles.)
Enc All know and supported image types.
Font Wrapper class for struct gdFont. There are only a fixed number of instances of these class, one for each built-in font in GD.
GDException Abstract base class for GD exceptions. These exceptions are intended to signal programmer errors and so ideally should be prevented instead of caught.
GDinvalidFormat Attempted to encode or decode an image to format 'Enc.UNKNOWN'.
GDinvalidImageData Attempted to use an ImageData as valid (i.e. containing valid data) when it wasn't. You should ensure ImageData.valid is true and that ImageData.type is not UNKNOWN before decoding an ImageData.
Image This class contains and manages GD image data.
ImageData Represents the contents of an image file encoded in one of the preferred formats. Can be read from or written to a stream and/or converted to an Image.
IMode Enumeration of all interpolation methods.
Point Represents a point on a plain. Supports integer or FP coordinates.
Rect Represents a quadrilateral, typically the bounding rectangle of a region of text. Since this rectangle may be rotated, there is no checking to ensure that this is actually a rectangle. This class is rarely used.
TrueRect A rectangle guaranteed to have all of its edges be horizontal or vertical.
